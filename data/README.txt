Data from connectivity analysis for "A generic decision-making ability predicts psychopathology in adolescents and young adults and is reflected in brain connectivity patterns that are distinct from general intelligence" (Michael Moutoussis and Benjamín Garzón, Sharon Neufeld, Dominik R. Bach, Francesco Rigoli, NSPN Consortium, Marc Guitart-Masip and Raymond J. Dolan)

melodic_IC.nii.gz: 200 node parcellation

bad_nets200.txt: list of nodes classified as artifacts

modules.nii.gz: modules (groups of nodes in the parcellation)

 vol0:  "SMT" # sensory/somatomotorsomatosensory + motor
 vol1:  "VIS" # visual
 vol2:  "MPC" # medial prefrontal cortex
 vol3:  "FPN" # frontoparietal control network
 vol4:  "FPL" # frontal pole
 vol5:  "OFC" # orbitofrontal cortex,medial and lateral
 vol6:  "RDC" # right DLPFC
 vol7:  "ATC" # anterior temporal cortex (including MTL)
 vol8:  "OPC" # opercular cortex
 vol9:  "PTC" # posterior temporal cortex
 vol10: "PCC" # posterior cingulate cortex
 vol11: "LDC" # left DLPFC
 vol12: "SUB" # subcortical
 vol13: "SAN" # salience network 

netmats_ridge.csv: connectivity matrices (1 per row)
